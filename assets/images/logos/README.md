# Logo Assets

This folder contains placeholder SVG logos for the IUSJC (Institut Universitaire Saint Jean du Cameroun) schools.

## Schools Included

1. **IUSJC Main Logo** (`sji-main.svg`)
   - Main institutional logo for Institut Universitaire Saint Jean du Cameroun
   - Website: https://iusjc.org/ or contact administration

2. **Saint Jean Ingénieur** (`iusjc.svg`)
   - Engineering school logo
   - Website: https://saintjean-ingenieur.com/

3. **Saint Jean Management** (`sjsm.svg`)
   - Business school logo
   - For official logo, contact: Saint Jean School of Management administration

4. **PrépaVogt** (`prepa-vogt.svg`)
   - Classes Préparatoires Intégrées (CPI)
   - Website: https://prepavogt.org/
   - Founded in 2006 by the Frères de la Congrégation Saint Jean (CSJ)

## Replacing Placeholder Logos

To replace these placeholders with official logos:

1. **Option 1: Contact the institutions directly**
   - Request official logo files (PNG, SVG, or high-res JPG)
   - Ensure you have permission to use the logos

2. **Option 2: Download from official websites**
   - Visit each school's official website
   - Look for press kit or brand assets section
   - Or right-click on logos and save

3. **Supported Formats**
   - SVG (recommended for scalability)
   - PNG (with transparent background)
   - WebP (good compression)

4. **File naming convention**
   Keep the same file names:
   - `sji-main.svg` or `sji-main.png`
   - `iusjc.svg` or `iusjc.png`
   - `sjsm.svg` or `sjsm.png`
   - `prepa-vogt.svg` or `prepa-vogt.png`

## Note About the Current Placeholders

The current SVG logos are stylized placeholders that represent:

- **Saint Jean Management**: Growth chart representing business excellence

These should be replaced with official institutional logos before production deployment.
